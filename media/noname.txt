Для тех кому интересно/любопытно.
В продолжении темы по изучению низкоуровнего Metal API от Apple, сделал для себя интересное открытие - API заточен под работу на системах с общей памятью (процессоры с 64х битной архитектурой, поколения А7 или моложе), т.е. CPU и GPU получают прямой доступ к одним данным без необходимости перебрасывать их по шине PCI, это позволяет смешивать произведение вычислений на GPU и CPU. Прирост производительности в отдельных случаях до х10. Самое итересное, что Metal API способен решать задачи не только с рэндерингом 3D контента, но и осуществлять Data-Parallel Compute Processing с чудовищной производитеьностью. 
Если накормить девелоперов достойным обучающим материаллом, это будет прорыв в архитектуре мобильных приложений. Существующая потребность в "серверных фермах" начнет приобретать устойчивую альтернативу в лице произведения расчетов на стороне миллиардов клиетских гаджетов.
Очень любопытно.
